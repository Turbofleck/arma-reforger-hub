Server-Einrichtung

Detaillierte Anleitungen zur Erstellung und Konfiguration Ihres dedizierten Arma Reforger Servers für jede Plattform.

PC Dedicated Server Setup

  1. Voraussetzungen: Stellen Sie sicher, dass Sie Arma Reforger auf Steam besitzen. Laden Sie die "Arma Reforger Server" App über das Tools-Menü in Ihrer Steam-Bibliothek herunter.
  2. Server-Verzeichnis erstellen: Erstellen Sie einen Ordner für Ihre Serverdateien (z.B. `C:\reforger-server`). Kopieren Sie die heruntergeladenen Server-App-Dateien in diesen Ordner.
  3. Basiskonfiguration (`config.json`): Erstellen Sie eine `config.json`-Datei im Hauptverzeichnis. Hier definieren Sie den Servernamen, das Passwort, die maximale Spielerzahl und die grundlegenden Admin-Rechte. 
    {
    "dedicatedServer": {
        "serverName": "Mein Crossplay Server",
        "password": "IhrSicheresPasswort",
        "maxPlayers": 32,
        "gameHostBindAddress": "0.0.0.0",
        "gameHostBindPort": 2001,
        "gameHostRegisterToBackend": true
    }
}
  4. Szenario-Konfiguration: Wählen Sie ein Szenario (z.B. Conflict oder Game Master) und konfigurieren Sie es. Dies geschieht oft durch eine weitere JSON-Datei, die das Szenario, die Fraktionen und Mod-Abhängigkeiten festlegt.
  5. Start-Skript erstellen: Erstellen Sie eine Batch-Datei (`.bat` für Windows) oder ein Shell-Skript (`.sh` für Linux), um den Server mit den richtigen Parametern zu starten. Wichtig ist der `-config` Parameter, der auf Ihre `config.json` verweist.
  6. Port-Freigabe: Geben Sie die in der `config.json` definierten Ports (standardmäßig UDP 2001 und 5001) in Ihrer Firewall und Ihrem Router frei, damit Spieler von außen beitreten können.
  7. Server starten: Führen Sie Ihr Start-Skript aus. Der Server sollte im Server-Browser von Arma Reforger erscheinen.

Konsolen Dedicated Server (via Nitrado)

Für Xbox und PlayStation ist die Anmietung eines Servers beim offiziellen Partner Nitrado der einzige Weg, einen dedizierten 24/7-Server zu betreiben.

  1. Nitrado-Konto erstellen: Besuchen Sie die Nitrado-Website und erstellen Sie ein Konto.
  2. Server bestellen: Wählen Sie Arma Reforger (Xbox/PlayStation) aus der Spieleliste. Konfigurieren Sie Ihren Server, indem Sie die gewünschte Spieleranzahl und Mietdauer auswählen.
  3. Basiskonfiguration über das Webinterface: Nach der Bestellung erhalten Sie Zugriff auf ein Webinterface. Hier können Sie grundlegende Einstellungen wie Servernamen, Passwort und Willkommensnachricht einfach per Mausklick ändern.
  4. Szenario und Mods auswählen: Das Webinterface bietet eine vereinfachte Methode zur Auswahl des Spielmodus (z.B. Conflict). Mods aus dem In-Game-Workshop können ebenfalls über eine ID-Liste hinzugefügt werden. **Achtung:** Nur Mods, die für Konsolen freigegeben und Crossplay-kompatibel sind, werden funktionieren.
  5. Server starten/neustarten: Nutzen Sie die Steuerelemente im Webinterface, um Ihren Server zu starten, zu stoppen oder neu zu starten. Änderungen an der Konfiguration erfordern in der Regel einen Neustart.
  6. Server finden: Ihr Server erscheint nach kurzer Zeit im Community-Server-Browser auf Ihrer Konsole. Suchen Sie nach dem von Ihnen vergebenen Namen.
